# Sandwi4 SMP

Static GitHub Pages website for the Sandwi4 SMP Minecraft server.

## Pages

- `index.html` - home page
- `server.html` - server information
- `rules.html` - rules
- `team.html` - staff
- `vote/index.html` - vote/server-list links, available as `/vote`
- `sandwi4-vault.html` - hidden admin panel

## Admin Login

Open `sandwi4-vault.html`.

- User: `owner`
- Password: `Sandwi4SMP!2026`

The site is built for GitHub Pages, so admin changes are stored in the browser with `localStorage`. For global live edits, update the files in the repository or connect a backend service.

## Publish On GitHub Pages

1. Push these files to a GitHub repository.
2. Open repository settings.
3. Go to Pages.
4. Select Deploy from branch.
5. Choose the branch and `/root`.
